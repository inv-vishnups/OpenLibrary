
# To create system architect

```
Use .cursor/agents/system-planner.mdc

Read the following project documents:

docs/product-spec.md
docs/database-schema.md
docs/features/

Generate the complete system architecture for the OpenLibrary project.

The plan must include:
- Prisma database schema
- NestJS backend architecture
- Angular frontend architecture
- Docker infrastructure

Save the result to:

docs/system-architecture.md
```

```
Use .cursor/agents/autonomous-dev.mdc

Using the architecture from docs/system-architecture.md and the product requirements from docs/product-spec.md, generate the entire OpenLibrary project including:

- Angular frontend
- NestJS backend
- Prisma schema
```
